Qr Code restaurant Project

Project tijdens de zomervakantie

Tijdens de zomervakantie was ik met een goede vriend op café in Mol en we wilden een drankje bestellen. We zagen een qr code op onze tafel liggen en we dachten dat we hier de menu kaart konden zien maar dit was de code voor de wifi. We vroegen aan de ober of we de menukaart konden bekijken maar blijkbaar hadden ze er geen.

Dit vonden we geen leuke beleving als klant en hadden plots het idee gekregen om een qr code te maken zodat je deze kan scannen als klant en hier ook bestellingen mee kan doen.

Uitwerking

Dit project hebben we met 3 vrienden op poten gezet. We hebben hier het python framework Django gebruikt. Wekelijks hebben we meetings gehouden dit online op discord en ook hebben we samen gezeten en hier aan gewerkt. We hebben tevens ook gecommuniceerd met een potentiële klant om feedback over onze applicatie te geven.
